# 
<h1>NMAP SCAN</h1>

<h2>Description</h2>
This project involves using the nmap tool to scan the target server and identify ports and services running on the machine.
<br />

<h2>Languages and Utilities Used</h2>

- <b>NMAP</b> 

<h2>Environments Used </h2>

- <b>Kali Linux</b>
